# Project Rules — uhomes-student-housing

## README 三语同步规则

本项目维护 3 个语言版本的 README，每次更新必须同步修改全部 3 个文件：

| 文件 | 语言 | 位置 |
|------|------|------|
| `README.md` | English | 项目根目录 |
| `README_CN.md` | 中文 | 项目根目录 |
| `README_JP.md` | 日本語 | 项目根目录 |

规则：
- 三个文件的内容结构和版本号必须保持一致
- 每种语言的 README 严格使用该语言，只有技术命令（`clawhub install`）和专有名词（uhomes、en-suite、studio、PBSA）保留英文
- 不要在中文 README 中混入英文描述，不要在日语 README 中混入中文
- 更新任何一个 README 时，必须同时更新其他两个
