# uhomes-student-housing

[English](./README.md) | [中文版](./README_CN.md)

[OpenClaw](https://openclaw.ai) と [Claude.ai](https://claude.ai) 向けの AI スキルです。留学生が [uhomes.com](https://en.uhomes.com) で認証済みの学生向け住居を見つけるのをサポートします。

**ClawHub** で公開中: [`uhomes-student-housing@1.5.1`](https://clawhub.ai/skills/uhomes-student-housing)

## インストール

```bash
clawhub install uhomes-student-housing
```

インストール後、自然な言葉で質問するだけで利用できます：

```
早稲田大学の近くで学生マンションを探しています
```

```
東京の学生向け住居の相場を教えてください
```

```
en-suite と studio の違いは何ですか？
```

## 仕組み

### 1. 意図を理解

都市の概要が必要か、物件検索か、部屋タイプの比較かを自動判断し、適切な回答を返します。

### 2. 二つのデータソースからリアルタイム取得

- **大学物件ページ**：uhomes.com から最新の物件データをリアルタイム取得
- **必住リスト**：uhomes編集チームが厳選した都市・大学別の優良物件ランキング

### 3. 四段階のスマート推薦アルゴリズム

```
第1段階  ハードフィルター  → 低評価・遠距離・予算超過の物件を除外
第2段階  スコアリング      → 距離（指数減衰）+ 予算適合度
                            + 必住リスト順位ボーナス + ベイズ品質評価
第3段階  スロット選択      → 5つの異なる観点からそれぞれ最適な物件を選出
第4段階  理由の生成        → 各推薦に「なぜこの物件か」のラベルを付与
```

### 4. 多角的な5件の推薦を表示

各推薦は異なる観点で最適化されており、似通った物件ばかりにならない多様な選択肢を提供します：

| スロット | 最適化の観点 | ラベル例 |
|---------|------------|---------|
| ⭐ おすすめ | 総合スコア最高 | 「総合評価トップ」 |
| 🏆 必住リスト | 必住リスト順位最上位 | 「必住リスト第1位」 |
| 📍 最寄り | キャンパスまで最短 | 「徒歩3分」 |
| 💰 最安値 | 価格最安 | 「£172/週」 |
| ❤️ 最高評価 | ベイズ調整後の評価最高 | 「4.6★（349件）」 |

### 5. 直接予約リンク

各物件に uhomes.com の直接予約リンクが付きます。末尾には必住リスト全体と検索ページへのリンクも掲載。

## 対応言語

日本語、英語、中国語に対応。ユーザーの言語を自動検出し、同じ言語で回答します。物件名や部屋タイプの用語（en-suite、studio、PBSA）は英語のまま表示されます。

## 必住リスト対応範囲

推薦アルゴリズムは uhomes 必住リストのデータを統合しており、**22大学**と**英国17都市 + ボストン**に対応。必住リスト掲載物件は推薦でボーナスを獲得し、🏆バッジが付きます。

## 対応都市

イギリス（ロンドン、マンチェスター、バーミンガム、エディンバラ、グラスゴー、オックスフォード、ケンブリッジ、ブリストル、リーズ、サウサンプトン、シェフィールド、リバプール、ダラム、ノッティンガム、コヴェントリー等）、オーストラリア（シドニー、メルボルン他6都市）、アメリカ（ニューヨーク、ボストン他6都市）、カナダ（トロント、バンクーバー、モントリオール）、日本（東京、大阪、京都）、アイルランド、シンガポール、香港。
