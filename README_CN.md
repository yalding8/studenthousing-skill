## 工作原理

### 1. 理解你的意图

自动判断你需要城市概况、具体搜索还是房型对比，给出对应的回答方式。

### 2. 双数据源实时获取

- **大学房源页**：从 uhomes.com 实时抓取在售房源
- **必住榜**：uhomes 编辑团队精选的城市/大学维度的优质房源排行

### 3. 四阶段智能推荐算法

```
第一阶段  硬性过滤  → 排除低评分、过远、严重超预算的房源
第二阶段  综合评分  → 距离（指数衰减）+ 预算匹配 + 必住榜加分 + 贝叶斯品质分
第三阶段  多维选位  → 从 5 个不同维度各选最优房源
第四阶段  生成理由  → 每个推荐附带为什么选它的标签
```

### 4. 展示 5 个差异化推荐

每个推荐占据一个独立维度——确保你看到的是多样化选择，而不是 5 个雷同的结果：

| 位置 | 优化维度 | 标签示例 |
|------|---------|---------|
| ⭐ 综合推荐 | 总分最高 | "最佳匹配" |
| 🏆 必住推荐 | 必住榜排名最高 | "必住榜第 1 名" |
| 📍 距离最近 | 步行时间最短 | "步行 3 分钟到校" |
| 💰 价格最优 | 价格最低 | "£172/周" |
| ❤️ 口碑最佳 | 贝叶斯修正后评分最高 | "4.6★（349 条评价）" |

### 5. 直达预订

每个房源附带 uhomes.com 直达预订链接，末尾附有完整必住榜和搜索页链接。
