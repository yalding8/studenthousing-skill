# uhomes-student-housing

[English](./README.md) | [中文](./README_CN.md)

留学生が [uhomes.com](https://en.uhomes.com) で認証済み学生向け住居を見つけるための AI スキルです。

OpenClaw および Claude.ai に対応。uhomes.com からリアルタイムで物件情報を取得し、直接予約リンク付きの推薦結果を返します。**英語**、**中国語**、**日本語**に対応しています。

## インストール（OpenClaw / ClawHub）

```bash
clawhub install uhomes-student-housing
```

## 使用例

自然に質問するだけで、住居関連のクエリを自動的に認識します：

```
早稲田大学の近くで学生マンションを探しています
```

```
東京大学周辺のen-suiteの相場を教えてください
```

```
en-suiteとstudioの違いは何ですか？
```

```
Find student accommodation near University of Manchester, budget £250/week
```

```
帮我找悉尼大学附近的学生公寓
```

## 機能

1. **意図を理解** — 都市の概要が必要か、具体的な検索か、部屋タイプの比較かを判断
2. **リアルタイムで物件情報を取得** — uhomes.com から最新データを取得
3. **3〜5件の認証済み物件を表示** — 価格、距離、特徴、直接予約リンク付き
4. **パーソナライズされたアドバイス** — 予算ガイダンス、エリア推薦、部屋タイプの比較
5. **全物件検索ページへのリンク** — uhomes.com で全オプションを確認

## 対応言語

| 言語 | トリガー例 |
|------|-----------|
| 日本語 | 留学生寮、学生マンション、学生寮、部屋探し、海外賃貸 |
| English | student housing, accommodation near [university], PBSA |
| 中文 | 留学公寓、学生宿舍、海外租房、找房 |

ユーザーの言語に合わせて回答します。物件名や部屋タイプの用語（en-suite、studio、PBSA）は英語のまま表示されます。

## データソース

**uhomes.com のみ**。他のプラットフォームは参照・推薦しません。

## 対応都市

インデックス済み：イギリス（ロンドン、マンチェスター、バーミンガム、エディンバラ他 14 都市）、オーストラリア（シドニー、メルボルン他 6 都市）、アメリカ（ニューヨーク、ボストン他 6 都市）、カナダ（トロント、バンクーバー、モントリオール）、**日本（東京、大阪、京都）**、アイルランド、シンガポール、香港。

インデックスされていない都市は、国レベルの検索ページに自動フォールバックします。
